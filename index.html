SkylishCSS it's a modern CSS framework that are a small project right now.

I'm working alone with it right now, but I letting it to be share able for everyone from the start. You can use this CSS framework for

  • Buttons > READY
  • Animation > NOT READY
  • Navigation > NOT READY
  • And more coming.

I have seperate the .css files because if your only want to choose the Buttons/Animations/Navigations only. But maybe in the future I going to merge all files togather when I have much more codes for it.

SkylishCSS -> Documentation -> Buttons

Buttons 1 Right / Left (r / l ) got different shapes.

id="sky-button-1-r"

id="sky-button-1-l"

If you want to change color ("Fixed Colors") you just writing like this.

Remember that you always need to have uppercase on the first letter = "Red", "Blue", "Green".

Right Shapes

href="#" id="sky-button-1-r color="Red"

href="#" id="sky-button-1-r color="Blue"

href="#" id="sky-button-1-r color="Green"

Left Shapes

href="#" id="sky-button-1-l color="Red"

href="#" id="sky-button-1-l color="Blue"

href="#" id="sky-button-1-l color="Green"

Right now I only got three different "Fixed Colors" Red,Blue,Green.

If you want to change background colors for the buttons you can use:

< style scoped > #sky-button-1-l {Background: black} </ style > But remember I using :hover so you need to change that too if your going to use < style scoped >
