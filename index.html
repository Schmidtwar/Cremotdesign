SkylishCSS it's a modern CSS framework that are a small project right now.

I'm working alone with it right now, but I letting it to be share able for everyone from the start. You can use this CSS framework for

  • Buttons > READY
  • Animation > NOT READY
  • Navigation > NOT READY
  • And more coming.

You just need to download the .css file and start using it. You can get the files from my github site.

How to use SkylishCSS?

When you have added the .css file into your directory you can only write in

< link rel="stylesheet" href="sky.button.css" type="text/css" />.

You can find all the codes from Documentation

Example...

If you want to add a button in the HTML file you just write

< a href="#" id="sky-button-4">Click Here< /a>

and the button is instantly complete with colors. If you want to add "Fixed colors" you just write in

href="#" id="sky-button-4" colors="Red"

Remember that you always need to have uppercase on the first letter when your adding the color "Red", "Blue", "Green". You can find all the colors for each buttons in the _Documentation _.
